Usage Examples

Minimal:
<ge-header .userInfo=${{ nom: 'Dupont', prenom: 'Jean', email: 'jean@ge.ch', typeCompte: 'PP' }}></ge-header>
With navigation menu:
<ge-header showMenu .userInfo=${'{...}'}></ge-header>
Constrained width:
<ge-header .fullWidth=${'{false}'} showMenu .userInfo=${'{...}'}></ge-header>
