Web Component Select Mes Espaces

Exemple

Utilisation

Les liens sont fournis dans un JSON avec l'url dans l'attribut href et le label dans l'attribut label.

Un séparateur horizontal peut être ajouté en remplaçant un lien par "--".

Code de l'exemple ci-dessus:

    <select-mes-espaces>
    links='[
        { "href": "https://example.com/apple",   "label": "Fiscales" },
        { "href": "https://example.com/banana",  "label": "Subventions culturelles" },
        "--",
        { "href": "https://example.com/cucumber","label": "Éducation" }
      ]'
    >
    </select-mes-espaces>
